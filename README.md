# Planejamento Estratégico de TI

Este repositório reúne os materiais e resumos das aulas do curso de Planejamento Estratégico de Tecnologia da Informação.

## Visão Geral

O objetivo do projeto é organizar conteúdos teóricos e práticos das aulas para facilitar consultas futuras e revisões. Cada pasta de aula contém documentos com análises, diagramas, resumos e materiais de apoio.

## Como Usar

1. Abra a pasta da aula desejada.
2. Leia `Material - XX` para o conteúdo principal.
3. Consulte `ResumoAula.md` para uma visão rápida dos principais pontos.
4. Use `Análise.md` e `Diagrama.md` para aprofundar conceitos e visualizar processos.

## Observações

- Alguns arquivos podem ficar em branco até serem preenchidos com notas ou exercícios.
- Esta organização facilita a preparação para provas e trabalhos acadêmicos.
